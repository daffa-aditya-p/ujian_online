JIKA ADA MASALAH
================================================================================

1. Reset Database:
   cd /workspaces/ujian_online/backend
   /usr/bin/php8.2 artisan migrate:fresh --seed --force

2. Clear Cache:
   cd /workspaces/ujian_online/backend
   /usr/bin/php8.2 artisan config:clear
   /usr/bin/php8.2 artisan cache:clear

3. Rebuild Frontend:
   cd /workspaces/ujian_online/ujian_online
   npm run build
